Day 01 - html의 기초와 태그

  • html의 구조 : html, head, body로 이루어짐
  • h1~h6 : 제목 만들기
  • p : 단락 만들기
  • br : 줄 바꾸기
  • hr : 수평줄 만들기
  • blockquote : 단락 만들기
  • table : table, tr, td, th를 이용한 표 만들기
    - colspan, rowspan : 행 또는 열 합치기
    - caption : 표에 제목 붙이기
  • img : 이미지 삽입하기
    - alt : 이미지 설명하기
    - width, height : 이미지 크기 조정하기
